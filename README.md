# MEAN-Stack
MongoDB, Express, Angular, Node.js

Basic Mean Stack CRUD.

## Requeriments
MongoDB (4.0.10)
Express (4.17.1)
Angular (8.1.1)
Node.js (10.16.0)

## Install
npm install on folder NodeJS
npm install on folder AngularApp

##  Run servers
nodemon serve on folder NodeJS
ng serve on folder AngularApp
